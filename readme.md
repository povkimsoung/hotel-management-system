# 🏨 Hotel Management System

Dear Teacher,

This is my final C++ project called **Hotel Management System**.

This project was developed using **Object-Oriented Programming (OOP)** concepts and file handling in C++. The system is designed to help manage hotel room records in a simple and user-friendly way.

## 📌 Project Features

- Add New Room
- Display Room Data in Table Format
- Search Room
- Sort Room By Price
- Delete Room
- CSV File Data Persistence
- Beautiful Table UI using Tabulate Library

## 💻 Technologies Used

- C++
- OOP Concepts
- Tabulate Library
- CSV File Handling
- Git & GitHub

## 🎯 OOP Concepts Used

- Class and Object
- Encapsulation
- Constructors
- Getters and Setters
- Functions

## 📂 Project Structure

- `main.cpp` → Main source code
- `hoteldata.csv` → Data storage file
- `README.md` → Project documentation
